This package contains the implementation specific classes for elliptic curve cryptography, which are not part of the SATSA specificiation.

This classes are also part of the Squawk class library, the corresponding package name is com.sun.squawk.security.ecc. As the Squawk classes are currently not visible they are duplicated in this package. In future versions of the library this package may be removed and the Squawk classes will be used directly.

The classes in this package are a slightly newer version than the corresponding classes in Squawk, the main difference is that a simple modulo exponentiation algorithm was added.

Furthermore do the Squawk classes not include the FiniteField class.
