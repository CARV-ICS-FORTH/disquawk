lookup_b

Operation

Look up value in sorted array

Format

lookup_b

Forms

lookup_b = 253 (0xfd)

Operand Stack

..., key, arrayref => ..., index

Description

This instruction is used to replace Java's lookupswitch. The array referenced by arrayref must be of type byte[] and in ascending order. If the key was not found in the array, the instruction returns -1.
