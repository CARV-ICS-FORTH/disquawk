A B C D E F G I L M N O P R S T U W X

findslot

Operation

Look up an interface method in an object's virtual method table

Format

findslot
index

Forms

findslot = 183 (0xb7)

Operand Stack

..., reference, class => ..., result

Description

This bytecode takes as input from the stack an object and an interface that object implements. The inline input is the index number corresponding to some interface method to be invoked. The instruction then looks up that method in the object's virtual method table and pushes the method's index number in the table onto the stack. This instruction and the invokeslot bytecodes serve as a replacement for the invokeinterface Java bytecode.
