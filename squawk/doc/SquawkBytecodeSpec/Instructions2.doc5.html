A B C D E F G I L M N O P R S T U W X

escape

Operation

Add 256 to following bytecode

Format

escape = 77 (0x4d) escape_wide_m1 = 78 (0x4e) escape_wide_0 = 79 (0x4f) escape_wide_1 = 80 (0x50) escape_wide_short = 81 (0x51) escape_wide_int = 82 (0x52)

Forms

escape = 77 (0x4d)

Operand Stack

See resulting bytecode description

Description

There are more than 256 Squawk bytecodes. The way bytecode instructions above 255 are represented is with the escape bytecode followed by something else. The various wide forms of the escape bytecode modify the resulting instruction the same way as the wide bytecode.

extend

Operation

Prepare method for execution

Format

extend
words

Forms

extend = 184 (0xb8)

Operand Stack

_ => _

Description

This instruction goes at the beginning of a method. The inline parameter denotes how many words should be zeroed out.

extend0

Operation

Prepare method for execution

Format

extend0

Forms

extend0 = 195 (0xc3)

Operand Stack

_ => _

Description

This instruction goes at the beginning of a method. It serves as shorthand for the extend bytecode with an inline parameter of zero.
