UEI-Compliant Squawk Emulator
Currently supported devices:
	Squawk-CLDC-1.1
	Squawk-IMP-1.0

Currently tested OSes:
	Mac OS X - x86
	WindowsXP

Currently tested IDEs:
	EclipseME
	NetBeans Mobility

To build the UEI compliant module, invoke the builder as:
	d uei [destination]
If no destination is specified, it defaults to uei/build
